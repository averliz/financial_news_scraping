CZ4045 - Natural Language Processing

## Installation Guide 
The `requirements.txt` file should list all Python libraries that 'news_scraping_yahoo.ipynb' Jupyter notebook depend on, and they will be installed using:

```
pip install -r requirements.txt
```

## Output 
This file will output 'news_data_1pp.json' file which stores all the news articles that we will be using for the project.
